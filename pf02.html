Actividad 08: Road to Hall of Fame (SQL Injection)

Marzo 2026

Indicaciones

Explotar vulnerabilidades SQLi In-band en entornos controlados de PortSwigger. Se requiere extraer información de bases de datos mediante ataques UNION, identificar versiones de sistemas (Oracle/MySQL) y realizar bypass de pantallas de login administrativo.

Explicación Técnica

Se implementaron ataques UNION SELECT para combinar resultados de tablas legítimas con datos sensibles. El proceso incluyó la determinación del número de columnas mediante valores NULL y la identificación de tipos de datos compatibles para exfiltrar nombres de usuario y contraseñas de la tabla users.

Actividad 09: Blind SQL Injection (Ataques Inferenciales)

Marzo 2026

Indicaciones

Recuperar información de bases de datos donde la aplicación no devuelve resultados directos. Se deben utilizar técnicas de respuestas condicionales, errores forzados y retardos de tiempo (Time delays) para reconstruir contraseñas carácter por carácter.

Explicación Técnica

Se utilizaron funciones como pg_sleep() en PostgreSQL para confirmar vulnerabilidades mediante el tiempo de respuesta. Para la extracción de datos, se inyectaron condiciones booleanas que permitieron inferir la contraseña del administrador basándose en la presencia o ausencia de contenido específico en la interfaz web.

Actividad 10: Evasión de Filtros y Vulnerabilidades XML

Marzo 2026

Indicaciones

Bypass de Web Application Firewalls (WAF) que bloquean palabras clave de SQL. Utilizar técnicas de codificación XML y ataques fuera de banda (Out-of-band) para exfiltrar datos cuando los canales convencionales están protegidos.

Explicación Técnica

Se empleó la codificación de entidades hexadecimales en etiquetas XML para evadir filtros de texto plano. Asimismo, se utilizó Burp Collaborator para capturar interacciones DNS/HTTP iniciadas por el servidor de la base de datos, permitiendo la exfiltración masiva de credenciales mediante canales externos.
