1. Establecimiento de Política Restrictiva

Como medida de endurecimiento (hardening) inicial, se definen políticas por defecto que descartan todo el tráfico entrante, saliente y de reenvío que no haya sido explícitamente autorizado.

BASH
iptables -P INPUT DROP
iptables -P OUTPUT DROP
iptables -P FORWARD DROP

2. Gestión de Estados: Conexiones Establecidas

Para permitir la fluidez de las comunicaciones ya iniciadas y autorizadas, se habilita el tráfico que posea un estado ESTABLISHED o RELATED.

BASH
iptables -A INPUT -m conntrack --ctstate ESTABLISHED,RELATED -j ACCEPT
iptables -A OUTPUT -m conntrack --ctstate ESTABLISHED,RELATED -j ACCEPT
iptables -A FORWARD -m conntrack --ctstate ESTABLISHED,RELATED -j ACCEPT

3. Tráfico DNS de Red Local a Internet

Se autoriza el tráfico DNS (puerto 53 TCP) originado en la red local (192.1.2.0/24) hacia cualquier destino externo.

BASH
iptables -A FORWARD -p tcp -s 192.1.2.0/24 -d 0.0.0.0/0 --sport 1024:65535 --dport 53 -m conntrack --ctstate NEW -j ACCEPT

4. Servidor de Correo: Tráfico Entrante y Saliente

Configuración para permitir la recepción y envío de correos electrónicos (puerto 25 SMTP) para el servidor ubicado en la IP 192.1.2.10.

Nota Técnica: Se utiliza el reenvío (FORWARD) asumiendo que el cortafuegos actúa como puerta de enlace perimetral.
ENTRANTE
iptables -A FORWARD -p tcp -s 0.0.0.0/0 -d 192.1.2.10 --sport 1024:65535 --dport 25 -m conntrack --ctstate NEW -j ACCEPT
SALIENTE
iptables -A FORWARD -p tcp -s 192.1.2.10 -d 0.0.0.0/0 --sport 1024:65535 --dport 25 -m conntrack --ctstate NEW -j ACCEPT

5. Servidores Web: Gestión de Tráfico HTTP

Habilitación de acceso público al servidor web interno (192.1.2.11) y autorización para que los clientes de la red local naveguen por Internet.

INTERNET -> WEB SERVER
iptables -A FORWARD -p tcp -s 0.0.0.0/0 -d 192.1.2.11 --sport 1024:65535 --dport 80 -m conntrack --ctstate NEW -j ACCEPT
RED LOCAL -> INTERNET
iptables -A FORWARD -p tcp -s 192.1.2.0/24 -d 0.0.0.0/0 --sport 1024:65535 --dport 80 -m conntrack --ctstate NEW -j ACCEPT
